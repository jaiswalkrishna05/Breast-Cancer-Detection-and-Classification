# Breast Cancer Detection and Classification using Deep Learning

Overview

Breast cancer is one of the most common forms of cancer worldwide, and early diagnosis plays an important role in improving treatment outcomes. This project focuses on building a deep learning-based system that can classify breast cancer images into two categories: benign and malignant.

The project uses Convolutional Neural Networks (CNN) along with Transfer Learning using ResNet50 to perform image classification. A simple web interface built with Streamlit allows users to upload histology images and receive predictions in real time.

The main goal of this project is to demonstrate how deep learning can be applied to medical image analysis and contribute to the development of intelligent healthcare solutions.

 Features

* Classification of breast cancer images into benign and malignant categories
* Implementation of both a custom CNN model and a ResNet50 transfer learning model
* Real-time predictions through a Streamlit web application
* Image upload support with probability-based output
* Performance evaluation using accuracy curves and confusion matrices
* User-friendly and lightweight interface

 Workflow

1. Collect and prepare the dataset.
2. Preprocess and augment the images.
3. Train the CNN and ResNet50 models.
4. Evaluate model performance.
5. Save the trained models.
6. Deploy the application using Streamlit.
7. Predict whether the uploaded image is benign or malignant.

 Output

The application predicts whether the uploaded histology image belongs to:
* Benign (Non-Cancerous)
* Malignant (Cancerous)
It also provides the probability score generated by the model.
